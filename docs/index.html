Global Universities Performance Overview

The chart on the webpage highlights the academic performance of the top 75 universities worldwide, focusing on their Teaching and Research scores. It illustrates the annual consistency of leading institutions. Use the dropbox to explore academic metrics from 2011 to 2024. To view the top universities in a specific country, select the country from the dropbox. You can use the reset button to go back to the initial state which displays the top 75 university rankings for all of the years combined. Hover over a data point for detailed information about each university.
